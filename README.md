# Teste Técnico QA

## Sobre o projeto
Este repositório contém a solução de um teste técnico de QA com três partes:

- Testes Web (Cypress)
- Testes de API (Cypress)
- Testes de Performance (JMeter)

---

## Tecnologias usadas
- Cypress
- JavaScript
- Apache JMeter
- Node.js

---

### Estrutura
cypress/e2e/web → testes web
cypress/e2e/api → testes de API
docs/bdd → cenários em BDD
jmeter → testes de performance# Teste Técnico QA
