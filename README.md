# Teste Técnico QA

## Sobre o projeto
Este repositório contém a solução de um teste técnico de QA com três partes:

- Testes Web (Cypress)
- Testes de API (Cypress)
- Testes de Performance (JMeter)

---

## Tecnologias usadas
- Cypress
- JavaScript
- Apache JMeter
- Node.js

---

## Diferenciais

- Estrutura organizada por tipo de teste
- Documentação em BDD
- Cobertura de cenários positivos e negativos
- Testes de performance com análise de resultados
