# Teste Técnico QA - Mirante

## Objetivo
Este repositório contém a solução do teste técnico de QA, contemplando três frentes:

- automação web
- automação de API
- testes de performance

## Tecnologias utilizadas
- Cypress
- JavaScript
- Apache JMeter
- Git / GitHub
